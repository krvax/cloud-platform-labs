# Cloud Platform Labs

Hands-on labs covering Cloud Infrastructure, SRE, and Platform Engineering.

## Topics

- **Kubernetes** — EKS cluster setup, deployments, troubleshooting, ArgoCD GitOps
- **Terraform** — VPC, IAM, ASG/ALB, remote state, OIDC federation
- **CI/CD** — GitLab CI, GitHub Actions, ArgoCD, pipeline design
- **Observability** — CloudWatch, monitoring concepts, log aggregation
- **Security** — Secrets Manager, IAM policies, OIDC
- **Python SRE** — Scripting, automation, coding exercises
- **AWS** — EC2, EKS, ECS, ALB, Route53, ElastiCache

## Structure

```
labs/           # Hands-on exercises (lab-01 through lab-15)
docs/           # Technical write-ups and concept deep-dives (EN/ES)
```
